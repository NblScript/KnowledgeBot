# 术语表

## 核心概念

| 术语 | 英文 | 说明 |
|------|------|------|
| 知识库 | Knowledge Base | 知识的容器，包含多个文档 |
| 文档 | Document | 知识的基本单元 |
| 分块 | Chunk | 文档切分后的片段 |
| 向量 | Embedding | 文本的向量表示 |
